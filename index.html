Semantic Elements

Semantic elements clearly define its content and are meaningful to both the browser and the developer. Semantic elements describe their meaning in a way that both a human and machine can understand. Semantic elements describe the purpose and the content accurately. When using HTML, semantic elements may be used to define the different parts of a web page. Semantic tags are used to enhance accessibility, searchability, internationalization, and interoperability. The main semantic tags for larger document outlines include <header>, <nav>, <aside>, <section>, <article>, and <footer>.

Using semantic elements allows for data to be reused and shared across different applications, enterprises, and communities. Semantic elements are much easier to read and understand for programmers, search engines, and assistive technologies which in turn makes it a better experience for users. Semantic elements can also help develop a more consistent code which makes it easier for everyone to understand.
